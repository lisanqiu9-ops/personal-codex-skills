# Style Patterns

Use these as inspiration only. Adapt the style to the user's content instead of copying a pattern wholesale.

## Precision Blueprint

Design Aesthetic: 干净、精密、受建筑蓝图与高端技术期刊启发的编辑风格。适合系统拆解、AI 架构、流程、数据平台和复杂方法论。

Background Color: 带轻微纸纹的灰白色 `#F8F7F5`。

Primary Font: Neue Haas Grotesk Display Pro 或思源黑体 Heavy。

Secondary Font: Tiempos Text 或思源宋体。

Color Palette: 主文本 `#2F3542`，主强调 `#007AFF`，辅助线条 `#B8C2CC`。

Visual Elements: 精细线框、坐标网格、剖面图、流程节点、编号标尺、少量高亮蓝色。

## Bold Beginner Guide

Design Aesthetic: 明亮、清晰、带一点玩具说明书感的教学风格。适合初学者教程、分步说明、培训材料和轻松科普。

Background Color: 温暖浅灰 `#FAF8F2` 或干净白色 `#FFFFFF`。

Primary Font: 思源黑体 Heavy 或 Inter Black。

Secondary Font: 思源黑体 Regular。

Color Palette: 主文本 `#1F2937`，主强调 `#FF6B35`，辅助强调 `#00A896`。

Visual Elements: 大编号、分步路径、手势箭头、模块化图块、简单图标、清楚的完成状态。

## Executive Evidence

Design Aesthetic: 克制、锋利、以证据为核心的商业编辑风格。适合战略汇报、行业洞察、投资材料和管理层阅读。

Background Color: 冷白 `#F6F7F9` 或深墨色 `#111827`。

Primary Font: Helvetica Now Display 或思源黑体 Heavy。

Secondary Font: Source Han Sans / Inter。

Color Palette: 主文本 `#111827`，主强调 `#C2410C`，辅助灰 `#6B7280`。

Visual Elements: 强标题、证据卡片、横向比较表、瀑布图、时间轴、稀疏但有力的数据标注。

## Technology Campus Competition

Design Aesthetic: 明亮、电影感、适合小学区级竞赛的科技创新展示风格。画面要像高质量活动主视觉：科技蓝舞台、阳光校园、中国红飘带、友好机器人和航天元素共同出现。

Background Color: 科技深蓝 `#063A78` 与校园暖白 `#FFFDF7` 组合，局部用阳光金 `#FFC857` 提亮。

Primary Font: 思源黑体 Heavy 或 Microsoft YaHei Bold。

Secondary Font: 思源黑体 Regular 或 Microsoft YaHei。

Color Palette: 主文本 `#FFFFFF` 或 `#102A43`，主强调 `#00AEEF`，精神强调 `#D71920`，温暖强调 `#FFC857`。

Visual Elements: 友好白色机器人、红领巾、中国红飘带、学校建筑、火箭/卫星、蓝色 HUD 面板、发光轨迹线、盾牌符号。适合图像生成模型先生成整页主视觉，再本地叠加准确中文文字。避免低清素材拼贴、假 Logo、随机英文和过度军事化。

## Archive Essay

Design Aesthetic: 档案馆、展览墙和长文杂志结合的叙事风格。适合历史、人物、文化、社会议题和研究综述。

Background Color: 温润纸色 `#F3EFE7`。

Primary Font: 方正兰亭黑或思源黑体 Bold。

Secondary Font: 思源宋体或 Noto Serif CJK。

Color Palette: 主文本 `#2B2A27`，主强调 `#8B1E3F`，辅助墨线 `#9A948A`。

Visual Elements: 档案标签、边注、引文块、时间刻度、扫描纹理、单色照片处理、证据墙式布局。
